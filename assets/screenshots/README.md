# Screenshots

Add screenshots here for use in:
- Chrome Web Store listing
- GitHub repository preview
- Marketing materials

## Recommended Screenshots to Add

| Filename | Content |
|----------|---------|
| `mpop-hero.png` | Extension popup showing project detected |
| `mpop-proposal.png` | Generated proposal in the extension |
| `mpop-settings.png` | Settings panel with API key |
| `mpop-landing.png` | Full landing page screenshot |
| `mpop-mobile.png` | Mobile view of landing page |

## Dimensions for Chrome Web Store
- Screenshots: 1280x800 or 640x400 pixels
- Extension icon: 128x128 pixels (already in extension folder)

## How to Take Screenshots
1. Open Chrome with MPOP installed
2. Go to a Freelancer.com project page
3. Click the MPOP icon
4. Use Chrome's built-in screenshot or Snipping Tool
