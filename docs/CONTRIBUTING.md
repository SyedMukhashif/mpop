# Contributing to MPOP

Thank you for your interest in contributing to MPOP by KurtSaz!

---

## How to Contribute

### 🐛 Reporting Bugs
1. Check if the bug is already reported in Issues
2. Email us at **we.kurtsaz@gmail.com** with:
   - What happened
   - What you expected to happen
   - Steps to reproduce
   - Your Chrome version
   - Screenshot if possible

### 💡 Suggesting Features
Send your feature ideas to **we.kurtsaz@gmail.com** with:
- What problem does this feature solve?
- How would it work?
- Who would benefit from it?

### 🔧 Code Contributions
1. Fork this repository
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Test thoroughly on Freelancer.com
5. Submit a Pull Request with a clear description

---

## Code Standards

### Extension (JavaScript)
- Use clear variable names
- Comment complex logic
- Test on both logged-in and logged-out Freelancer pages
- Handle all error cases explicitly
- No external libraries — keep it lightweight

### Landing Page (HTML/CSS/JS)
- Keep it as a single self-contained HTML file
- Test on mobile (375px), tablet (768px), desktop (1440px)
- All interactive elements must work without JavaScript gracefully
