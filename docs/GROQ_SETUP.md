# Getting Your Free Groq API Key

Groq is the free AI service that powers MPOP's proposal writing.
It's completely free — no credit card, no payment required.

---

## Step-by-Step

### 1. Go to Groq Console
Open your browser and go to:
```
https://console.groq.com
```

### 2. Sign Up
Click **"Sign Up"** — you can use:
- Google account (easiest)
- GitHub account
- Email + password

### 3. Create an API Key
Once logged in:
1. Click **"API Keys"** in the left sidebar
2. Click **"Create API Key"**
3. Give it a name (e.g. "MPOP Extension")
4. Click **Create**
5. **Copy the key immediately** — it starts with `gsk_...`

> ⚠️ Save this key somewhere safe. Groq only shows it once.

### 4. Add to MPOP
1. Click the MPOP icon in Chrome toolbar
2. Click **⚙ Settings**
3. Paste your key in the **"Groq API Key"** field
4. Click **Save Settings**
5. Done! You're ready to generate proposals.

## Free Tier Limits

| Limit | Amount |
|-------|--------|
| Requests per day | 500 |
| Requests per minute | 30 |
| Tokens per minute | 6,000 |
| Cost | $0 forever |

**500 requests/day is more than enough** for even the most active freelancers.
At 10 proposals per day, your key will last for 50 days without hitting limits.

---

## Troubleshooting

**"Invalid API key" error:**
- Make sure there are no spaces before or after the key
- The key must start with `gsk_`
- Try creating a new key at console.groq.com

**"Rate limit" error:**
- You've hit the per-minute limit — wait 60 seconds and try again
- This is rare unless generating many proposals very quickly
