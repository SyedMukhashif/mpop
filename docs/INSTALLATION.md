# MPOP Extension — Installation Guide

## Step-by-Step for Chrome

### Step 1 — Download the Extension
Download `MPOP_by_KurtSaz.zip` from the `extension/` folder in this repository.

### Step 2 — Extract the ZIP
- **Windows:** Right-click the ZIP → "Extract All" → Choose a folder (e.g. Desktop)
- **Mac:** Double-click the ZIP file — it extracts automatically

### Step 3 — Open Chrome Extensions
Open Google Chrome and type this in the address bar:
```
chrome://extensions
```
Press Enter.

### Step 4 — Enable Developer Mode
In the top-right corner of the Extensions page, toggle **Developer Mode** to ON.

### Step 5 — Load the Extension
Click **"Load unpacked"** button (appears after enabling Developer Mode).
Navigate to and select the extracted `sadia_extension` folder.

### Step 6 — Confirm Installation
You should see the **MPOP by KurtSaz** extension appear in your list.
Click the puzzle piece icon in Chrome toolbar → pin MPOP so it's always visible.

## Getting Your Free Groq API Key

See [GROQ_SETUP.md](GROQ_SETUP.md) for the full guide.

**Quick version:**
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up with Google or email (free)
3. Click **API Keys** → **Create API Key**
4. Copy the key (starts with `gsk_...`)
5. Open MPOP extension → ⚙ Settings → paste key → Save

## First Use

1. Go to any project page on Freelancer.com
   - URL looks like: `freelancer.com/projects/...`
2. Click the MPOP icon in your Chrome toolbar
3. You'll see the project title and details auto-detected
4. Choose your **Tone** and **Length**
5. Click **Generate Proposal**
6. Review the proposal, copy it, paste it into Freelancer!

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Extension not showing | Make sure Developer Mode is ON and folder is correctly selected |
| "Could not read page" | Refresh the Freelancer project page and try again |
| API error: Invalid key | Check your Groq key starts with `gsk_` — get a new one at console.groq.com |
| Rate limit error | Wait 60 seconds — Groq free tier allows 500 requests/day |
| No project detected | Make sure you're on a `freelancer.com/projects/` page |
